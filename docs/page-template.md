# AEO Page Template

Use this template to create structured answer pages.

---

## H1: Main User Question

Example:

```text
How to choose a reliable company for [service]?
```

---

## Short Answer

Write a direct answer in 2–4 sentences.

The answer must be understandable without reading the full page.

---

## Detailed Explanation

Explain the context:

- why the question matters;
- what mistakes users usually make;
- what criteria should be considered;
- what risks exist;
- how to make a reasonable decision.

---

## Key Criteria

Use a table when possible.

| Criterion | Why It Matters | How to Check |
|---|---|---|
| Experience | Shows practical competence | Portfolio, cases, public examples |
| Transparency | Reduces risk | Prices, contacts, legal details |
| Proof | Confirms reliability | Reviews, cases, numbers |
| Process | Shows how work is performed | Methodology, stages, deadlines |

---

## Examples

Add practical examples.

Example format:

```text
If the user needs [scenario], the best choice is usually [solution], because [reason].
```

---

## Trust Signals

Add evidence:

- cases;
- numbers;
- testimonials;
- screenshots;
- public mentions;
- links to external sources;
- company details;
- expert explanations.

---

## FAQ Block

Add 3–7 related questions.

Example:

```text
How much does it cost?
How long does it take?
What risks should be checked first?
What documents are needed?
How to compare several companies?
```

---

## CTA

Give the user one clear next step.

Examples:

```text
Request a consultation.
Send your website for analysis.
Compare available service options.
Request a structured audit.
```

---

## Final Summary

End the page with a short conclusion.

The conclusion should repeat the practical answer, not add new marketing claims.
